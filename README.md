# Subscription Form

A simple subscription and payment selection form built with HTML, CSS, and JavaScript.

## What it does

- Lets the user check a subscription checkbox
- Lets the user pick a payment method — Visa, MasterCard, or PayPal
- On clicking submit, shows whether the user subscribed and which payment method they chose
- Handles the case where no payment method is selected

## Files

| File | Purpose |
|------|---------|
| `index.html` | Structure of the form |
| `style.css` | Styling and layout |
| `index.js` | Form logic on submit |

## How to run

No setup needed. Just open `index.html` in your browser.

```bash
# clone the repo
git clone https://github.com/PRUDHVI-1627/Subscription.git

# open the file
cd Subscription
open index.html
```
